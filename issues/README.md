# Issues - Vấn đề cần Giải quyết

Nơi lưu các vấn đề đã phát hiện, cần bàn thêm hoặc giải quyết.

## Khác biệt với /ideas
- **/ideas** = Ý tưởng mới, tính năng muốn thêm
- **/issues** = Vấn đề hiện tại, cần sửa/cải tiến

## Cách sử dụng
- Mỗi vấn đề là 1 file `.md` riêng
- Đặt tên theo format: `YYYY-MM-DD-ten-van-de.md`
- Ghi rõ: Vấn đề là gì, tại sao quan trọng, hướng giải quyết (nếu có)

## Độ ưu tiên
- **critical** - Ảnh hưởng nghiêm trọng, cần giải quyết sớm
- **high** - Quan trọng, nên xử lý
- **medium** - Cần làm nhưng không gấp
- **low** - Nice to have

## Trạng thái
- **open** - Đang mở, chưa giải quyết
- **investigating** - Đang tìm hiểu
- **in-progress** - Đang xử lý
- **resolved** - Đã giải quyết
